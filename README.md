#voting_system
This project is a web-based voting system built with PHP, MySQL, HTML, CSS, and JavaScript. It supports user authentication, role-based access, candidate management, and secure voting with one vote per registered user.
Key Features
User registration and login
Admin dashboard for candidate and voter management
Voter dashboard for casting votes
One vote per user enforcement
Real-time vote tally and results display
Candidate registration with basic validation
MySQL database setup and table creation via db.php
Built-in admin account initialization
Admin Capabilities
Add, edit, and delete candidates
View all registered users
Manage voter accounts
View all votes
User Capabilities
Register as a voter
Log in securely
Vote for a candidate
View election results
Tech Stack
PHP
MySQL
HTML/CSS
JavaScript
Use this system as a foundation for a lightweight election or polling application and customize it further for production-ready features like stronger validation, vote auditing, and UI improvements.
